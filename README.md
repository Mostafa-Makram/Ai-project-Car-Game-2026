# Ai-project-Car-Game-2026
🚗 AI Smart Car Game

A 2D intelligent car game inspired by classic mobile games like Subway Surfers, enhanced with Artificial Intelligence techniques for decision-making and adaptive gameplay.

🎯 Project Overview

This project is a 2D multi-lane car simulation game where the player or AI controls a vehicle that must:

Follow traffic rules 🚦
Avoid collisions 🚗💥
Survive as long as possible ⏳

The game includes multiple modes and uses AI algorithms to improve behavior and difficulty dynamically.

🧠 AI Concepts Used

This project applies several AI techniques:

Model-Based Reflex Agent
Goal-Based Agent
Utility-Based Agent
🔍 Algorithms:
Heuristic Evaluation
Hill Climbing
Minimax + Alpha-Beta Pruning
Genetic Algorithm (GA)
